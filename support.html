Frequently Asked Questions

Q: Why isn't the Fn key working?

A: The Mac App Store version of Capy Voice operates in a secure "sandbox" environment, which restricts it from monitoring global function keys like the Fn key for privacy and security reasons. Instead, please use the global hotkey Control+Option+R to start/stop recording.

Q: How do I access Settings?

A: You can toggle the settings window by pressing Command+Shift+S at any time.

Q: Does it work offline?

A: Capy Voice uses advanced AI models (OpenAI/Gemini) which currently require an active internet connection to process your speech with high accuracy.
